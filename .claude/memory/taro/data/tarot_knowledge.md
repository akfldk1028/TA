## Fool's Journey 3막 구조

- **1막 (0-VII): 의식 세계** — 외부 권위와 세상의 힘 (Fool→Chariot)
- **2막 (VIII-XIV): 무의식 세계** — 내면 탐구 (Strength→Temperance)
- **3막 (XV-XXI): 초의식 세계** — 자아 초월과 통합 (Devil→World)

## 스프레드 포지션 상세 의미

### Celtic Cross 10장
1. **Present**: 현재 상황의 핵심
2. **Challenge**: 장애물 또는 통합할 에너지 (항상 부정적이진 않음)
3. **Past**: 상황의 깊은 뿌리, 무의식적 기반
4. **Future**: 다가오는 에너지 (수 주 내)
5. **Above**: 의식적 목표, 열망, 최선의 결과
6. **Below**: 무의식적 동기, 숨겨진 감정
7. **Advice**: 조언, 취해야 할 태도
8. **Environment**: 주변 사람/환경의 영향
9. **Hopes**: 희망과 두려움 (종종 동전의 양면)
10. **Outcome**: 현 궤도의 종합적 결과

## 해석 원칙

### 원소 상호작용 (Elemental Dignities)
- Fire+Air (Wands+Swords): **우호적** — 서로 강화
- Water+Earth (Cups+Pentacles): **우호적** — 서로 강화
- Fire+Water (Wands+Cups): **적대적** — 서로 약화
- Air+Earth (Swords+Pentacles): **적대적** — 서로 약화
- 같은 원소: **증폭** — 에너지 2배 (좋든 나쁘든)
- Major Arcana: 원소 초월, 닿는 것을 증폭

### Reversed 해석 원칙 (프로젝트 기준)
1. **차단된 에너지**: 의미는 있으나 표현이 막힘
2. **내면화**: 외부 대신 내면에서 경험
3. **약화/과잉**: 정방향의 약한 버전 또는 왜곡된 과잉
4. **절대 재앙 예언 금지** — 성장과 내면 작업의 초대로 프레이밍

### 패턴 읽기
- 같은 suit 다수: 해당 영역에 초점
- Major Arcana 다수: 영혼 수준의 중요성
- 같은 숫자 반복: 해당 테마 강조
- 연속 숫자: 자연스러운 진행/시퀀스

## 프로젝트 데이터 불일치 주의

`tarot_data.json` 이름과 `tarot_card.dart`의 `_majorArcana` set 불일치:
- `"The Papess/High Priestess"` → set에는 `"The High Priestess"`
- `"The Pope/Hierophant"` → set에는 `"The Hierophant"`
- `"The Wheel"` → set에는 `"Wheel of Fortune"`
- Minor arcana는 소문자 (`"ace of wands"`) vs UI는 타이틀 케이스 기대

`tarot_api.json` — Waite 원전 설명 + 의미 (별도 데이터 소스, 현재 미사용)
